# Study Buddy Hub 

Study Buddy Hub is a simple web application designed to help students stay organized by managing courses, tasks, and study-related resources in one place.

##  Features
- Course pages for different subjects
- Task tracking to stay on top of assignments
- Clean and simple user interface
- Organized file structure with HTML, CSS, and JavaScript

## Built With
- HTML
- CSS
- JavaScript

##  Project Structure
- `index.html` – Main landing page  
- `homepage.html` – User dashboard  
- `courses.html` – Course overview  
- `tasks.html` – Task management  
- `signup.html` – User sign-up page  
- `CSS/` – Stylesheets  
- `js/` – JavaScript files  
- `assets/` – Images and other assets  

##  Purpose
This project was built as a learning-focused web development project to practice front-end development and project organization.
## Tech Stack
- **Frontend:** HTML, CSS, JavaScript  
- **Backend / Services:** Firebase Authentication, Firebase Firestore  
- **Development Tools:** VS Code, Git, GitHub  
- **Languages (in progress):** Java, SQL  

## Features
- User authentication (sign up & login) using Firebase  
- Personalized homepage after login  
- Course tracking for enrolled classes  
- Task management to organize study work  
- Navigation system with back buttons for all pages  
- Responsive and beginner-friendly UI
